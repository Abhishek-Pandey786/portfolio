# Premium Developer Portfolio

This project is a premium, story-driven developer portfolio built with Next.js, Tailwind CSS, and modern motion.

## Quick Start

```bash
npm run dev
```

## Update Content

Edit the structured data in [data/portfolio.ts](data/portfolio.ts) to replace placeholder content with your real information:

- Name, location, and tagline
- Social links and coding profiles
- Experience, projects, research, and certifications
- Resume highlights and availability

## Visual Assets

Add your real assets to the `public/` folder and update references as needed:

- Profile photo
- Project banners
- Resume PDF
- OpenGraph preview image

## Scripts

- `npm run dev`: Start the development server
- `npm run build`: Build for production
- `npm run start`: Run the production build
- `npm run lint`: Lint the project
